\## Overview

This ETL connector fetches IP geolocation data from \[freegeoip.app](https://freegeoip.app) using \*\*three endpoints\*\*:

\- `/json/{ip}`

\- `/csv/{ip}`

\- `/xml/{ip}`



It transforms the data into JSON format and stores it in MongoDB.
