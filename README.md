# 🔍 VirusTotal File Hash → MongoDB ETL Connector

This project is a Python **ETL (Extract → Transform → Load)** pipeline that:
1. **Extracts** malware analysis data from the VirusTotal API (using specified file hashes)
2. **Transforms** it (adds ingestion timestamps)
3. **Loads** it into a MongoDB collection for storage and analysis.
